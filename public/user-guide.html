🚀 5-Minute Quickstart

  1. Import cases → IMPORT XLS → Select Excel → Done
  2. Check today → Left panel shows hearings by date
  3. Open a case → Click party name (blue link) → Case file opens

Chapter 2: Adding Data

3 ways to add cases:

Method 1: New Entry

New Entry form: Court, Judge, Type, Case No.*, CNR, Plaintiff*, Defendant, Next Date, Our Side, Task

Steps: Click NEW ENTRY → Fill form → Click ADD. Required: Case No. and Plaintiff.

Method 2: Import Excel

EXPORT XLS and IMPORT XLS buttons in the right panel menu

Excel format: Sr, Court, Judge, Type, Case No., Parties, Next Date, Fixed For, Previous Date, CNR No.

Import result dialog showing added cases and errors

Importing creates NEW entries. To edit existing cases, use the Case Overlay.

Method 3: Export

Click EXPORT XLS → File saves as Caseline_Cases.xlsx → Edit in Excel → Re-import if needed.

Chapter 14: eCourts Sync

Cases tab: view linked cases with CNR, sync individually, see changes (date/stage updates in green banners)

4 Tabs: Cases | CNR Dump | Report | Settings

How to sync: Case needs CNR → Click 🔄 Sync → System opens eCourts, fills CNR, solves CAPTCHA, downloads data (30-60 sec)

Bulk CNR Dump: paste multiple CNR numbers (one per line), click Parse & Match to link to existing cases

Settings: What to Download, Schedule (auto-sync time/strategy/max per day), Sync Mode (Browser free / API paid), Conflict Resolution, Notifications
